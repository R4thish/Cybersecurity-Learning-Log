# Study Journey 

This repository documents my continuous learning journey in **Cybersecurity, Networking, Linux, Programming, and related technologies**.

## Repository Structure

- `Daily_Logs/` – Day-by-day record of what I studied and practiced.
- `Notes/` – Short topic-wise notes.
- `Lab_Practices/` – Write-ups from hands-on labs (TryHackMe, OverTheWire, etc.).
- `Scripts/` – Small utility scripts I write while learning.
- `Projects/` – Mini projects and experiments.

## Goals

- Build strong fundamentals in cybersecurity and networking  
- Maintain consistent daily practice  
- Create a public record I can share with recruiters and mentors  
- Track my progress over time
